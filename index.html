reveal-plantuml

A reveal.js plugin that allow using PlantUML diagrams

Features

  • Configurable PlantUML server address
  • PlantUML diagrams on Markdown slides
  • PlantUML diagrams on HTML slides
  • CDN support (unpkg, jsDelivr)
  • NPM package
